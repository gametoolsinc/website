How to calculate the number of Stacks in Minecraft?

You can calculate the number of Stacks by dividing the number of Single Items you have by 64 and then rounding your answer down to the nearest whole number. The number that you just calculated is how many Stacks you have! You can then calculate the number of Single Items remaining by subtracting the number of Stacks multiplied by 64 from the total number of Single Items. If you follow these steps, you can easily calculate how many Stacks a given number of Single Items is.
