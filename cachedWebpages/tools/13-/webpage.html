Introduction

Online NBT editor

NBT files are binary files which are difficult to read for humans. NBT files however are crucial to store data in Minecraft. To easily create and edit this data, there is the Online NBT editor. The program handles all binary data back-end, so you don’t have to worry about it. The process is simple and uncluttered, making editing of NBT files easier than ever.

Question

What is a NBT file?

Named Binary Tag (short as NBT) is a Minecraft file format that contains in-game data. It is used to store big amounts of binary data, without taking up more space than needed. In Minecraft it is for example being used as the data for maps and schematics.

Question

Why use the Online NBT editor?

Using the Online NBT editor, you can edit and save NBT files. These files can later be used to spice up your Minecraft world. The Online NBT editor features a clean and structured user interface, to make editing the files as simple as possible. Simple click on the text and edit the values.

Example

How to use the Online NBT editor?

Upload your NBT file using the input field on top of the editor. After uploading your file, it will automatically be loaded into the editor. Simply click on some text or number to edit its value, or click on the type-indicating icons to change the binary type. You can also add or remove new elements by clicking on the type-indicating icon and clicking on 'new' or 'delete'. When you are done, simply press ‘Save’ in the top right and download your file.
